AColumn

class apandas.AColumn(name: str, func: AFunction | Callable | Any | None = None, override: bool = False)

Just a named function that can be accessed (and constructed on-the-fly if needed) from an AFrame. No check is done that the shape conforms the DataFrame (hence can be added). Cached in the frame on the calculation.
