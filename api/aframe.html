AFrame

class apandas.AFrame(*args, verbose=False, **kwargs)

AFrame is a subclass of pandas.DataFrame that allows to use AColumn objects as keys and calculate them on their definition on the fly. All the keys are actually strings, but the AFrame modifies pd.DataFrame methods so they handle the conversion to strings and possibly creating the columns internally.

So most of pd.DataFrame methods are actually modified via the metaclass - in particular, AColumns in *args and **kwargs of method calls are converted to strings and the columns are created if they don’t exist. In addition, if the return is a pd.DataFrame, it is converted to AFrame.
